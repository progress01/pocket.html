# Lite-Pocket使用說明

## 專案簡介
基於單一 HTML 檔案開發的個人稍後閱讀管理與筆記工具。不需要安裝資料庫或後端伺服器，所有資料皆儲存在您的瀏覽器本地端（LocalStorage）。
https://progress01.github.io/pocket.html/

## 核心功能
1. 智慧收藏
輸入網址後系統會嘗試自動抓取文章標題與內文。若因網站阻擋導致抓取失敗，系統會自動建立卡片並引導使用者手動貼上內容，確保資料不遺漏。

2. 閱讀模式
提供無干擾的閱讀介面，並內建三種模式：
速讀模式：適合快速瀏覽，提供簡易筆記欄。
精讀模式：提供康奈爾筆記法架構（關鍵字、筆記、總結）與目錄導航。
英文模式：整合單字卡製作、AI 提示詞複製、語音朗讀與字典查詢功能。

3. 行動裝置優化
針對手機瀏覽器進行了介面重構。在手機上閱讀時，版面會自動調整為垂直排列（上方顯示文章，下方固定顯示筆記區），解決了舊版版面擠壓的問題。

4. 資料管理
支援將所有文章與筆記匯出為 JSON 格式備份，或將單篇文章匯出為 HTML 講義。亦可匯出 Anki 通用的 CSV 單字卡。

## 使用方式
將 index.html 檔案下載後，直接使用 Chrome、Safari 或 Edge 等瀏覽器開啟即可。

注意事項
由於瀏覽器的安全性限制，若您直接在電腦資料夾中點擊開啟檔案（file 協定），剪貼簿自動貼上功能可能會受限。建議將此檔案部署至 GitHub Pages 等支援 HTTPS 的環境，以獲得最完整的操作體驗。
